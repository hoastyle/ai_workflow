## 🎯 按功能分类

### 代码理解和搜索

| 工具 | 服务器 | 说明 |
|-----|--------|------|
| `semantic_search` | Serena | 语义搜索代码片段 |
| `semantic_query` | MorphLLM | 语义查询代码库 |
| `repo_query` | MorphLLM | 仓库查询和搜索 |
| `grep_search` | MorphLLM | 全文搜索 |

**推荐组合**:
1. 先用 `semantic_query` 定位文件
2. 再用 `grep_search` 精确搜索
3. 最后用 `morph_file` 编辑
